Sobre PyMat

PyMat é um projeto de extensão do departamento de Matemática da UFRPE que tem como proposta reunir três tópicos fundamentais para uma formação acadêmica, multidisciplinar e tecnológica: programação, matemática e ciências. Exploraremos, de forma prática, temas fundamentais da matemática e de outras áreas relacionadas usando a poderosa linguagem de programação Python.

Python é uma linguagem de programação gratuita e conhecida por ser fácil de aprender, ler e escrever. Neste sentido, propicia facilidade de acesso e aprendizagem, enquanto que a grande quantidade de pacotes para uso científico disponibilizados fornece versatilidade. Atualmente, Python é adotado como primeira linguagem de programação do currículo de vários cursos de exatas das melhores universidades do mundo.

O evento destina-se aos alunos de graduação, pós-graduação e docentes que buscam uma ferramenta científica para estudo, trabalho e pesquisa acadêmica, mas não necessariamente desejam fazer um curso completo de programação de computadores.

Ecossistema Python para matemática computacional

Palestra #1 On-line
Durante a última década, Python tornou-se indiscutivelmente a principal linguagem de programação para pesquisas científicas exploratórias, interativas e baseadas em computação. Esta palestra apresenta as vantagens do Python para a pesquisa científica e apresenta várias das principais bibliotecas e ferramentas Python usadas na Matemática e em outras áreas de conhecimento.

Edgar Amorim

Python: gráficos e outras visualizações

Workshop #1 On-line
Em muitos casos, a visualização de gráficos e objetos matemáticos torna-se uma ferramenta indispensável para analisar comportamentos e padrões de funções, equações e dados científicos. Neste workshop, usaremos o Python e sua extensão matemática NumPy para construir e customizar gráficos e objetos matemáticos em duas e três dimensões .

Python e problemas matemáticos desafiadores

Workshop #2 On-line
Neste workshop apresentaremos uma série de problemas matemáticos que exigirão mais do que apenas insights matemáticos para serem resolvidos. Embora a matemática o ajude a chegar a métodos elegantes e eficientes, o uso de um computador e habilidades de programação serão necessários para resolver a maioria dos problemas. Neste caso, usaremos a linguagem de programação Python para resolvê-los.

10:00 - 12:00

Maité Kulesza

Algumas abordagens para a modelagem da epidemia de COVID-19

Palestra #3 On-line
Com a pandemia da COVID-19, passamos a ouvir muitos termos como crescimento exponencial, achamento de curva, imunidade de rebanho, e também a perceber o quanto a modelagem matemática é importante para que possamos fazer prognósticos que nos ajudem a tomar decisões no presente e “modificar” o futuro. No entanto, realizar esses avaliações não seria possível sem o uso de modelos computacionais na epidemiologia. Essa palestra pretende apresentar um pouco mais do jargão envolvendo modelagem de epidemias, bem como, trazer alguns dos estudos computacionais que têm sido feitos e, em especial, trazer um experimento feito com alunas e e uma professora do Ensino Médio que puderam iniciar, com Python, a modelagem da COVID-19 em Pernambuco utilizando dados do Ministério da Saúde.

14:00 - 16:00

Ricardo Machado

Criptografia RSA em SageMath/Python

Workshop #3 On-line
Criptografia é um assunto tão antigo quanto a civilização humana. Estudaremos o algoritmo RSA, nomeado em homenagem aos seus inventores Rivest, Shamir e Adleman. Este algoritmo descreve um sistema de criptografia de chave pública, os detalhes de como o sistema funciona são completamente públicos e mesmo assim, após mais de 30 anos ele ainda não foi quebrado. Os conceitos matemáticos serão apresentados e consistem na teoria dos números básica. Um algoritmo será implementado em SageMath e em seguida faremos adaptações para que ele funcione em Python.

19:00 - 21:00

Edgar Amorim

Construção de imagens e simulações dinâmicas com Python

Workshop #4 On-line
Por ser um disciplina abstrata, muitas vezes se faz necessário a utilização de diferentes tipos de mídia para desenvolver conteúdos de matemática em sala de aula. Neste workshop vamos utilizar Python para construir imagens e animações. Vamos ver como exibir conjuntos de Julia e de Mandelbrot, e também como visualizar a dinâmica das soluções de alguns sistemas de equações diferenciais.

10:00 - 12:00

Adenilton Silva

Computação Quântica

Palestra #2 On-line
A computação quântica é um novo paradigma de computação que permite, por exemplo, realizar fatoração em tempo polinomial e a busca em dados desordenados com um ganho quadrático no tempo. Os primeiros computadores quânticos estão em desenvolvimento e o computador quântico (Sycamore do Google) executou em segundos um benchmark que requer dias para ser executado em um supercomputador. Microsoft, IBM, Google e diversas outras empresas possuem computadores quânticos em desenvolvimento. Nesta palestra serão apresentados os conceitos básicos da computação quântica.

14:00 - 17:00

Filipe Lima

Manipulação de dados e regressões lineares com Python

Workshop #5 On-line
"O recurso mais valioso do mundo não é mais o petróleo, e sim, informação". Nesse workshop o estudante vai ter uma noção introdutória do pacote Pandas. Esse pacote permite que usuário, carregue, manipule e visualize dados utilizando a linguagem Python. Como exemplo mais importante, usaremos dados da FAO sobre safras anuais de trigo, arroz, milho e soja de diversos países e faremos regressões lineares utilizando esses dados.
