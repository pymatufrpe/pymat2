Sobre PyMat

PyMat é um projeto de extensão do departamento de Matemática da UFRPE que tem como proposta reunir três tópicos fundamentais para uma formação acadêmica, multidisciplinar e tecnológica: programação, matemática e ciências. Exploraremos, de forma prática, temas fundamentais da matemática e de outras áreas relacionadas usando a poderosa linguagem de programação Python.

Python é uma linguagem de programação gratuita e conhecida por ser fácil de aprender, ler e escrever. Neste sentido, propicia facilidade de acesso e aprendizagem, enquanto que a grande quantidade de pacotes para uso científico disponibilizados fornece versatilidade. Atualmente, Python é adotado como primeira linguagem de programação do currículo de vários cursos de exatas das melhores universidades do mundo.

O evento destina-se aos alunos de graduação, pós-graduação e docentes que buscam uma ferramenta científica para estudo, trabalho e pesquisa acadêmica, mas não necessariamente desejam fazer um curso completo de programação de computadores.

Julio Lombaldo (UFRGS)

A importância da Programação na Carreira em Exatas

Palestra #1 On-line
Atualmente a matemática, bem como os demais cursos de exatas, tem diversos caminhos ainda inexplorados na sua totalidade quando se trata de programação. Linguagens de programação estão cada vez mais presentes e de fácil compreensão. O uso dessas como ferramentas no aprendizado e ensino de matemática, física e engenharias em geral tem se mostrado cada vez mais eficaz e necessária. Durante o começo da pandemia iniciei um projeto dedicado a estudar e ensinar usando ferramentas de programação que anteriormente só usava para pesquisa. Os resultados advindos desse projeto foram além das expectativas e o intuito dessa palestra é compartilhar possíveis novos caminhos de carreiras em exatas aliados a programação em geral.

Francisco Bustamante (IFRJ)

Streamlit - seu código Python na web em poucos passos

Palestra #2 On-line
Usualmente dedicamos muito tempo no desenvolvimento de códigos para resolver um determinado problema. Mas como mostrar visualmente o resultado para as partes interessadas, especialmente as que não lidam tanto com programação? E, mais, de uma forma fácil, interativa e com o mínimo de barreiras? Durante muito tempo, especialmente em alguns nichos mais científicos, esse papel foi exercido pelo Jupyter Notebook. No entanto, cada vez mais há a necessidade de adotar meios de compartilhamento mais práticos e multiplataformas. Afinal, nada mais simples que clicar em um link para uma página web. Nesse contexto, o Streamlit tem se destacado. Nesta palestra, veremos exemplos de aplicativos e entenderemos a estrutura básica de um projeto Streamlit.

A confirmar (Alura)

Primeiros passos em Machine Learning

Workshop #3 On-line
Uma das definições clássicas do Machine Learning diz que "[...] é um campo de estudo que dá aos computadores a habilidade de aprender sem terem sido programados para tal." Durante a palestra, vamos entender um pouco mais sobre essa área da tecnologia que, nos últimos anos, vem ganhando cada vez mais notoriedade tanto no campo acadêmico quanto no mercado de trabalho - provavelmente você já ouviu falar de uma empresa data driven, né? Discutiremos também os problemas que conseguimos resolver com os tipos de aprendizado supervisionado e não supervisionado, bem como abordar as técnicas mais utilizadas denominadas classificação, regressão e clustering.

19:00 - 21:00

Leon Silva (UFRPE)

Algoritmos matemáticos: uma abordagem usando Python

Workshop #4 On-line
Desde o advento dos computadores, a importância dos algoritmos aumentou constantemente em todos os campos da matemática. Vamos apresentar não só algoritmos e problemas algorítmicos, mas também sua fundamentação matemática e a implementação usando Python. Entre os algoritmos que serão abordados estão os método de Sturm (localizar zeros de polinômios), Runge Kutta (resolve EDO’s numericamente) e Gram-Schmidt (ortogonização de vetores).
